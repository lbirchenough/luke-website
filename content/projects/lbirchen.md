# lbirchen.com

## Overview

A modern Next.js site showcasing projects and an auto-updating resume. The app renders Markdown resume content fetched from GitHub, supports rich formatting, and exposes an API route that serves the latest PDF generated by a GitHub Actions workflow in the resume repository. The resume content is stored in a GitHub repository with Actions workflows that build PDF and HTML versions, and the site fetches and displays the latest versions dynamically.

## Tech Stack

- **Frontend:** Next.js 15 (App Router), React 19, Tailwind CSS 4, CSS modules, Lucide Icons
- **Hosting:** Vercel

## Features

- Markdown resume fetched dynamically from a separate GitHub repository
- API route that serves the latest PDF generated by a GitHub Actions workflow
- Project cards with expandable detail sections
- Project detail pages rendered from local markdown files (this page!)

## Architecture

The resume lives in a separate GitHub repo with Actions workflows that build PDF and HTML versions on every push. The portfolio site fetches and renders the latest markdown at request time, keeping the resume always up to date without redeploying the site.
