# Workopia

## Overview

A full-stack job board application where employers can post job listings and job seekers can search, bookmark, and apply to positions. Features user authentication, job search functionality, bookmark management, and application tracking with role-based authorization. Built following Laravel best practices including database migrations, Artisan seeders for test data generation, and Eloquent ORM for clean model relationships and queries. Built with Vite for asset compilation.

## Tech Stack

- **Frontend:** Blade templates, Tailwind CSS 4.1.17, Alpine.js
- **Backend:** Laravel 12.0, PHP 8.2+
- **Database:** PostgreSQL (dev), MySQL (production), Eloquent ORM with migrations, seeders, factories
- **Hosting:** Digital Ocean via Laravel Forge

## Architecture

Built following Laravel best practices:

- Database migrations, Artisan seeders, and factories for test data generation
- Eloquent ORM for clean model relationships and queries
- Role-based authorization for employer vs job seeker actions
- Vite for asset compilation

## Features

- User registration and authentication
- Job listing creation and management
- Search and filter functionality
- Bookmark management
- Application tracking
